Project Overview

I began by planning a professional networking quiz that combined multiple choice and true/false questions, provided immediate feedback, and included a tip after each answer. I drafted ten questions, answers, and tips in a Word document before starting development.

The original quiz used outdated HTML and JavaScript, so I refactored it into a modern, accessible version. I replaced the table-based layout with semantic div containers, moved styling to CSS, and updated the UI for clarity and readability. Navigation buttons and automatic feedback for correct or incorrect answers were added.

The final quiz is professional, user-friendly, and accessible, blending clean design, intuitive interactivity, and educational content to support users in learning networking skills.
