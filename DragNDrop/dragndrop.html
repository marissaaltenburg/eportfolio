Project Overview

I redesigned a legacy drag-and-drop activity to help users match skills with the six Holland Career Types, modernizing both functionality and design. I began by analyzing the original HTML, CSS, and JavaScript, then drafted new content to ensure the activity was relevant, intuitive, and educational for users exploring career interests.

I updated the visual design and feedback system, adding emoji-based responses for correct (✅) and incorrect (❌) matches, and created a dynamic progress bar to provide users with a sense of accomplishment. The layout was restructured with flex containers and button-based draggable items, improving readability, responsiveness, and accessibility. I also refreshed the color scheme, typography, and hover states to create a professional, business-oriented aesthetic.

JavaScript was refactored to integrate these new features while maintaining the drag-and-drop functionality. This required careful troubleshooting and iterative testing to manage legacy code, event listeners, and DOM updates. I also implemented a randomized skill order and a completion message to reinforce learning and user engagement.

The result is a cleaner, more accessible, and engaging drag-and-drop activity that blends modern design, intuitive interactivity, and educational value, providing users with a professional and enjoyable learning experience.
