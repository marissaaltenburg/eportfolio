Project Overview

I created a professional networking quiz combining multiple choice and true/false questions with immediate feedback and quick tips. I refactored outdated HTML and JavaScript into a modern, accessible design using divs, CSS, and improved UI.

Navigation buttons, automatic feedback, clear spacing, and readable fonts were added. The final quiz is professional, intuitive, and engaging, helping users learn networking skills effectively.
